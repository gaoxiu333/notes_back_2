
## 导航

[React Navigation](https://reactnavigation.org/)

## 常用API

- `navigation`
	- `navigate` - 导航
		- `navigate('Page')`
		- `navigate('page',{screen:'SubPage'})` - 导航至嵌套的子路由
- `Tab navigation` 
	- 自定义样式
		- 背景色 
			- 导航内容背景色
			- 导航背景色
		- 配置 icon

## ICON

- [React Native Vector Icons](https://github.com/oblador/react-native-vector-icons)
	- 支持在导航等元素中直接使用
