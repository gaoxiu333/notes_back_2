# 必要的基础支持和配置说明

### 安装

```bash
npm install --global rollup
```

### 运行

```bash
rollup main.js --file bundle.js --format iife
# or npm script
```

> 如果使用配置文件`rollup.config.js`,可以指定`--config`或者`-C`制定。
>
> 如果不传文件名会按照以下顺序加载配置文件：
>
> rollup.config.mjs -> rollup.config.cjs -> rollup.config.js
