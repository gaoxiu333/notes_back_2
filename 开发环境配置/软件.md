#### 然后

然后是 Microsoft 的 Visual Studio Code

## 开发环境配置

### VSCode

安装：

```bash
brew install visual-studio-code
```

配置：

登陆 `github` 账号同步配置

### Iterm2

安装：

```bash
brew install iterm2
```

安装完后切换 `Minimal` 主题

### nodejs

使用`fnm`管理node版本
使用 corepack 安装 pnpm

```bash
corepack enable
corepack disable
```
